Here are the steps I took to create this project.

- `cddev`
- `cd htmx`
- `mkdir htmx-flask`
- `cd htmx-flask`
- `python -m venv .venv`
- `. .venv/bin/activate`
- `pip install Flask`
- `pip install requests`
- Create the file `server.py`.
- Start the server by entering `./run`
- Browse localhost:5000
